# Walmart-Stock-Analysis
Analyzing Walmart's stock trends from 2018 to 2022 (pre and post covid). Examiming strategic impacts and investor insights.

# Key Insights

1. Stock Performance:
 - Long-term growth with stable performance.
 - 50-day moving average reflects investor confidence.

2. Impactful Events:
 - COVID-19 pandemic-driven spikes.
 - Strategic e-commerce investments like Flipkart acquisition.

# Model Performance and Regression Insights

1. Key Metrics:
 - High Adjusted R-Square indicates strong predictive accuracy for the model.
 - Close price predictions align closely with observed trends.

2. Key Plots:
 - Residual plots for high, low, and close prices highlight prediction errors.
 - Line fit plots demonstrate how well the model fits the data for key variables.
